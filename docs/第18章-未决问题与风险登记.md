## 18.2 风险登记(v0.3 现状)

### 风险 R1 · 核心机制可能不够好玩

- **描述**:拆/合汉字作为战斗核心,是全新机制,存在"想着有趣、玩着无聊"的可能。
- **影响**:致命——核心不成立则项目不成立。
- **应对**:阶段 0 纸面原型 + 阶段 1 战斗原型尽早验证;设 go-no-go 闸门,不通过则停止或转向。
- **状态**:待解,首要风险

### 风险 R2 · 拆合交互在触屏 / 鼠标上可能笨重

- **描述**:拆/合若操作繁琐、像查字典,会劝退玩家。PC 鼠标与移动端触屏的最优交互不一定相同。
- **影响**:高。
- **应对**:砚台隐喻 + 实时合字提示系统(见第 4、12 章 v0.3);阶段 1 在 PC + 触屏两端都打磨手感。
- **状态**:待解

### 风险 R3 · 认字门槛劝退部分玩家

- **描述**:生僻字会让汉字知识较弱的玩家受挫。
- **影响**:中。
- **应对**:锁定常用字;每字标注拼音与释义;把"识字"包装成寓教于乐卖点。
- **状态**:可控

### 风险 R4 · 美术工作量(尤其字怪)对个人开发的压力

- **描述**:即便有"字即画"减负,字怪与背景仍需 AI 出图与人工整理。
- **影响**:中。
- **应对**:"字即画"砍掉大半插画量;字怪做成统一 prompt 家族;无帧动画。
- **状态**:可控

### 风险 R5 · 四端发布的工程与审核成本 ~~(v0.3 已关闭)~~

- **状态**:**已关闭**(v0.3)。微信/抖音小游戏移除,平台收敛到 Steam + iOS + Android 三端,审核成本与平台 API 复杂度大幅下降。

### 风险 R6 · roguelike 平衡难度高

- **描述**:多流派 + 多法宝的 build 平衡极难调。
- **影响**:中。
- **应对**:首发收敛到 3 流派;数值守住比例关系而非绝对值;Steam Playtest 看流派分布数据持续调。
- **状态**:待解

### 风险 R7 · 小游戏赛道竞争与买量成本 ~~(v0.3 已关闭)~~

- **状态**:**已关闭**(v0.3)。不再进入小游戏赛道。

### 风险 R8 · v0.2 养成层平衡 ~~(v0.3 已关闭)~~

- **状态**:**已关闭**(v0.3)。养成层全面移除,双系统平衡问题不复存在。

### 风险 R9 · 服务端工程对个人开发者的负担 ~~(v0.3 已关闭)~~

- **状态**:**已关闭**(v0.3)。Premium 模式下服务端从「刚需」降级为「可选」,主要靠平台 SDK(Steamworks / Game Center / Play Games)解决排行榜与云存档。

### 风险 R10 · 国内合规与审核 ~~(v0.3 已关闭)~~

- **状态**:**已关闭**(v0.3)。不在国内运营,实名 / 防沉迷 / 版号 / 概率公示均无适用范围。

### 风险 R11 · Steam Wishlist 不够,首发埋没(v0.3 新增)

- **描述**:Steam Premium 游戏 70%+ 销量产生在首 90 天,首日推荐位由 Wishlist 数 + 商店页热度决定。如果上架时 Wishlist < 5000,几乎拿不到任何首页曝光,游戏即使本身好玩也会埋没。
- **影响**:高。
- **应对**:商店页**早开放**(不晚于阶段 3 中段);Steam Playtest、Reddit、Discord、X、小红书、知乎多渠道持续推;参加 Steam Next Fest(每年三次,免费曝光);找 Streamer / KOL 关键节点曝光。**这是 Premium 模式下最重要的非技术工作**。
- **状态**:待解,关键风险

### 风险 R12 · 海外社区运营对个人独立开发者的精力负担(v0.3 新增)

- **描述**:Discord 服务器、Steam 讨论区、X 帖子、Reddit AMA、邮件反馈——上线后社区维护是新的全职工作量。处理不好会拖累开发节奏。
- **影响**:中。
- **应对**:**主战场只选 1~2 个**(推荐 Discord + Steam 讨论区);其余只发布关键节点公告;设公开邮箱代替 1:1 私信;开发期每周固定 4~6 小时社区时段,不每天碎片回应。
- **状态**:待解

### 风险 R13 · 简体中文 only 在 Steam 的非华人受众面窄(v0.3 新增)

- **描述**:Steam 全球玩家中,简中是主要语种之一(中国 + 海外华人 + 中文学习者),但绝对值与英语 / 日语 / 西语相比仍小。非华人玩家因语言劝退会限制销量天花板。
- **影响**:中。
- **应对**:首发就这样接受;**预留 i18n 框架**(文本与数据分离),后续 v1.x 阶段如果首发成功,可以补英文 / 日文本地化作为"扩大受众"的延续运营动作。**短期不投入本地化资源**。
- **状态**:待解

### 风险 R14 · 退款率拉高首发口碑(v0.3 新增)

- **描述**:Steam 持有 14 天内 + 游玩 2 小时内可无理由退款。如果新手前 2 小时体验有问题、内容承诺与实际不符、bug 多,退款率 >15% 会显著拖低 Steam 评分与算法排名。
- **影响**:高(可能比纯差评更致命)。
- **应对**:阶段 3 Steam Playtest 期间**专门追踪"前 2 小时游玩信号"**;首发前 bug 清零;新手引导打磨到位;商店页文案不夸大;**情愿低调宣发也不要虚高预期**。
- **状态**:待解,关键风险
