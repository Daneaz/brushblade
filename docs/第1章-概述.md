---

## 1.1 一句话定位

> 你扮演「正字者」,通过**拆解与重组汉字**来战斗,在程序生成的关卡中一路斩除「错别字妖」,修复正在崩坏的文字之界。

一句话电梯陈述:**「汉字版的肉鸽卡牌构筑——你的牌就是字,你的构筑就是拆字与合字。」**

## 1.3 目标受众

- **核心**:Steam 平台肉鸽卡牌构筑爱好者(Slay the Spire / Balatro / Inscryption 用户),有耐心、愿意为完整体验付费、看中"build 涌现感"。
- **延展**:海外华人玩家(Steam 国区用户为主)、对汉字文化感兴趣的非华人玩家、中文学习者。
- **关键人群特征**:与小游戏玩家相反,Steam 用户**愿意花 1~2 小时上手新机制**,看重内容厚度、玩法深度与完成度。这一约束直接决定了新手引导可以放缓、难度阶梯需要做厚(见第 11 章 v0.3)。

---

## 1.4 核心卖点

1. **独一无二的核心机制**——把汉字的「拆字 / 合字」造字法变成战斗的 build 系统。只有中文能做,且没人认真做过。
2. **部首即属性**——借汉字部首天然编码的语义,玩家靠直觉就能读懂战斗,零学习成本。
3. **文化质感**——水墨书卷美学;每个字附拼音与释义,寓教于乐,这是卖点而非负担。

---

## 1.5 设计支柱

所有设计决策都应回归这四条支柱来检验。

| 支柱 | 含义 |
|---|---|
| 拆合即玩法 | 战斗的核心动词是拆字与合字;一切系统挂在拆合规则引擎上 |
| 部首即语义 | 用造字法天然编码的意义,作为属性与战斗效果的来源,降低认知成本 |
| 文化质感 | 水墨、书法、成语、识字——文化元素不是皮肤,是玩法本体 |
| 构筑的深度 | 五流派 + 十跨属性组合 + 法宝,提供长线可重玩的 build 空间 |

---

## 1.6 目标体验

玩家在一局中应依次体验到:

1. **构筑的爽感**——把零散的部件重组成强力的字。
2. **取舍的紧张**——拆还是留、攒还是出,每个决策都有机会成本。
3. **滚雪球的高潮**——后期 build 成型时一套连招清场。
4. **发现的惊喜**——撞出一个跨属性组合、合出一个没见过的字。

而在多局之间,玩家应感到自己在"一路读上去、把汉字一个个正回来"——重复被赋予了意义(见第 2 章叙事框架)。

---

## 1.7 竞品定位

《字·斗》处在两类成熟产品之间的空白地带:

- **肉鸽卡牌构筑类**(如《杀戮尖塔》式):提供 build 深度与重玩性,但与中文文化无关。
- **成语 / 文字类小游戏**:文化贴近、受众广,但玩法多为消除或答题,深度有限。

《字·斗》取两者之长:**用肉鸽构筑的深度,承载汉字文化的独特机制**。它对个人开发者也友好——核心是逻辑与数据(适合 Claude Code),美术以"字即画"大幅降本(见第 12 章)。
