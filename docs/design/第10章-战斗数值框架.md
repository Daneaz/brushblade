## 10.1 基础框架

脱离框架的数字没有意义,先立框架。

- **玩家生命**:起始 50(可经 meta / 事件成长)。
- **行动点(AP)**:每回合 3 点。拆 = 1,合 = 1,出基础字 = 1,出高阶字(焱/燚/剑/海/念等)= 2。AP 不跨回合保留。
- **字库容量**:8 张。新合成 / 拾取的字若超出需替换或丢弃(可被法宝调整)。
- **部件池容量**:12 个。跨战斗保留,run 结束清空。
- **部件掉落**:每回合开始,部件池 +2 随机部件(池满则不掉)。随机池的属性权重为待设计项(纯随机 / 偏向玩家已有属性,阶段 1 实测定)。
- **敌人 HP 基准**(对标 5 章节,使玩家伤害"并跑"):

| 阶段 | 杂兵 HP | 小 Boss HP |
|---|---|---|
| 第一章 | 8~15 | ~40 |
| 第三章 | 20~35 | ~90 |
| 第五章 | 40~60 | — |
| 终 Boss「讹」 | — | ~300(四阶段,每阶段 ~75~80) |

> **v0.3 变更**:原"v0.2 养成层的数值缩放"整块删除。Premium 模式下无养成、无字卡/法宝升级、无角色,所有数值在一局之内由字本身、法宝本身、流派资源决定,跨局只在难度阶梯调整(见第 11 章 v0.3),没有"养成 buff"叠加。这让战斗数值平衡显著简化——只需平衡一套基线 + N 套难度阶梯变异规则。

### 10.1.1 五行生克结算(v0.4)

> 规则唯一来源:[wuxing-reference](wuxing-reference.md)。

- **相克**:克制方 ×1.5,被克方 ×0.5;心系中立,对所有属性 1.0x。口诀:木克土、土克水、水克火、火克金、金克木。
- **相生**:字的配方原料属性**去重**后含相生有序对 → 该字效果 **×3**;多对不叠乘。口诀:木生火、火生土、土生金、金生水、水生木。
- **叠算**:相生与相克**相乘**,理论最高 ×4.5。例:焚(木生火)基础 18 → 54;对金怪 → floor(18×3×1.5) = 81。
- **本章字表数值均为"基础值"**,不含生克乘数;乘数在战斗结算时套用。
- ×3 为 demo 实测的爽感值,列为 Playtest 平衡观察项(第 18 章 R6)。

## 10.2 五大资源系统

五条流派各挂一个专属资源,但跑在同一框架上。

| 资源 | 流派 | 规则 |
|---|---|---|
| 灼烧 | 燎原(火) | X 层 → 回合末造成 X×2 伤害,然后 −1 层 |
| 护甲 / 破甲 | 锋锐(金) | 护甲 X = 每次伤害 −X(最低 0);破甲永久降护甲;斩杀线 ≤ 阈值则处决 |
| 水势 | 江海(水) | 上限 10;每点 +10% 治疗;清空可「泻」造成 水势×4 全体伤害 |
| 召唤 + 生长 | 森罗(木) | 单位每回合生长 +1 级,每级 +2 攻;场上 ≥4 单位触发额外生长;场上上限 5 |
| 心魔 | 摄心(心) | 上限 10;≥3 每回合自伤=层数;≥5 半数概率不行动;=10 崩溃(跳过回合 + 自伤 心魔×3 + 清零) |

## 10.5 战例

### 战例一:燎原流(终 Boss 阶段,80 血)

设玩家字库有 燃、炽、焱,部件池有 火×2,持有法宝「星火燎原」。

- **回合 1**:炽(灼烧结算 2→3)+ 燃(挂 3 层灼烧)+ 合炎。回合末灼烧 3×3 = 9。Boss 80→71。
- **回合 2**:炎(12 + 2 层灼烧)→ 71→59;拆字得火×2;合燚。回合末灼烧 4×3 = 12。Boss 59→47。
- **回合 3**:燚(22,带灼烧触发星火燎原翻倍 → 44)→ 47→3;火部件直出 4 → 击杀。

### 战例二:锋锐流(终 Boss 阶段,80 血,10 护甲)

设玩家字库有 锋、削、刮、剑、刃、斩,法宝 锋芒毕露 + 势如破竹(斩杀线 16)。

- **回合 1 · 破甲投资**:锋(破 5,护甲 10→5)+ 削(破 3 → 4 伤被 2 甲减为 2)+ 刮(破 2,护甲 →0)。Boss 80→78,护甲 0。
- **回合 2 · 收割**:护甲 0,锋芒毕露生效,所有攻击翻倍。剑 20×2 = 40 → 78→38;刃 7×2 = 14 → 38→24。
- **回合 3 · 处决**:刃 14 → 24→10;10 ≤ 16,斩 → 直接处决。

两种完全相反的玩法(AOE 爆发 vs 单体狙杀),跑在同一组底层数字上——框架健壮性已验证。

## 10.6 平衡哲学

数值是首版基准,真实值必然在实测中调整。要守住的不是绝对值,而是几条比例关系:

- 升阶链每阶约 1.5~2 倍跳。
- 灼烧的单回合总量,中期应接近一个杂兵的血量。
- 终结连招的翻倍倍率,要让 2~3 回合恰好打完一个 Boss 阶段。
- 破甲流的"破甲投资期"应约 1~1.5 个回合;护甲清零后的爆发 DPS 要明显高于燎原(它牺牲了 AOE 和续航换来的)。
- **相生 ×3 与相克 ×1.5 的叠算摆幅(最高 4.5 倍)是首要观察项**:Playtest 重点看含相生配方的字是否碾压同阶无相生字、以及被克流派对局是否绝望。

**调的是这些比例,不是孤立的数字。**
