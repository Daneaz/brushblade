## 三、成语 Boss(墨海 51 层起,程序生成)

- 墨海的 Boss 从成语库随机生成,四字 → 四阶段套用统一数值模板(首字均衡/次字坚壁/三字强攻/末字狂攻),逐字五行由配置指定。
- **当前 6 条占位**(用户筛选终稿后替换,详见《成语Boss候选筛选表》229 条待筛):
  刀山火海(金土火水)/山崩海啸(土土水心)/冰天雪地(水心心土)/烈火干柴(火火心木)/飞沙走石(心水土土)/气吞山河(心心土水)
- **形象设计提示(事实层面)**:这类 Boss 数量开放(终稿 30~50 条),逐只出稿不现实——需要的是一套**按字生成的视觉规则**(字形 + 五行色 + 阶段状态的组合系统),而非单体立绘。设计时建议以此为一个「形象系统」课题。

## 四、补充事实

- 敌人在战斗中的现有呈现:字牌格(名字/当前 HP/攻/能力进度标记),受击有弹跳/飘字;换阶段有破阶飘字。
- 五行主题色(现行 UI):火朱砂红 / 水青蓝 / 木翠绿 / 金赭金 / 土褐 / 心中性灰蓝(`Theme.ElementColor`)。
- 层段背景色(形象所处环境):字林竹绿 / 词渊黛蓝 / 文山赭石 / 墨海墨青。
- 深度缩放只改数值不改形象:同一只怪在 1 层与 40 层外观一致。
