# 五行规则规格

> v1.0 · 2026-07-05 收口(原「五行参考文档」升格为正式规格)
> 本文档是战斗属性生克的**唯一规则来源**,第 3 / 6 / 8 / 10 章与本文档保持一致。
> 历史说明:本规格源自浏览器 demo 实测(2026-06-07 wuxing-fix-earth 修正),demo 代码已随仓库重置移除,规则以本文档为准。

## 属性集

六属性 = **真五行(木、火、土、金、水)+ 心**。

- 生克圈只含五行,采用经典口诀,玩家零学习成本。
- **心系中立**:不参与生克,与所有属性互为 1.0x。心系机制(心魔)随摄心流 DLC 设计。
- **土**:真实属性(部件 / 字 / 字怪都存在),但不做完整流派——定位为跨流派「防御 / 护盾」工具字家族(见第 10 章 10.3.6)。

## 相克

口诀:木克土,土克水,水克火,火克金,金克木

| 攻方 | 克制对象 | 伤害倍率 |
|------|---------|---------|
| 木 | 土 | 1.5x |
| 土 | 水 | 1.5x |
| 水 | 火 | 1.5x |
| 火 | 金 | 1.5x |
| 金 | 木 | 1.5x |

被克方对克制者伤害 0.5x。心系不参与相克,与所有系均为 1.0x。

## 相生

口诀:木生火,火生土,土生金,金生水,水生木

**触发规则(2026-07-05 收口):**

1. 取字卡配方的原料属性集合并**去重**。
2. 集合中存在至少一个相生**有序对**(母→子)→ 该字效果 **×3**。
3. **多对不叠乘**:无论配方含几对相生,至多 ×3。
4. 相生 ×3 与相克 ×1.5 **相乘**,理论最高 ×4.5。
5. 第 10 章字表数值均为**基础值**,生克乘数在战斗结算时套用。

例:

- 淋(氵+林,水+木)→ 水生木 → 基础 8 伤 → 实战 24。
- 焚(林+火,木+火)→ 木生火 → 基础 18 AOE → 54;对金系再 ×1.5 → floor(18×3×1.5) = 81。
- 壁(辟+土,金+土)→ 土生金 → 基础护盾 8 → 24。

> **平衡观察项(挂第 18 章 R6)**:×3 是 demo 实测的爽感值;与相克叠算后摆幅达 4.5 倍,Playtest 阶段重点观察含相生配方的字是否碾压同阶无相生字。
