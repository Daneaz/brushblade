# 五行参考文档

## 五行相克（本期已实现）

口诀：木克土，土克水，水克火，火克金，金克木

| 攻方 | 克制对象 | 伤害倍率 |
|------|---------|---------|
| 火 | 金 | 1.5x |
| 水 | 火 | 1.5x |
| 木 | 土 | 1.5x |
| 金 | 木 | 1.5x |
| 土 | 水 | 1.5x |

被克方对克制者伤害 0.5x。心系不参与相克，与所有系均为 1.0x。

## 五行相生（本期已实现）

口诀：木生火，火生土，土生金，金生水，水生木

**触发规则：** 字卡配方中任意两个部件/字的属性构成相生对，效果×3。与相克叠算。

| 母 | 子 | 代表角色 |
|----|----|----|
| 木 | 火 | 焚（木木火）→ AOE 54伤 |
| 火 | 土 | —（本期暂无此配方角色）|
| 土 | 金 | 壁（土金）→ 护盾24 |
| 金 | 水 | —（本期暂无此配方角色）|
| 水 | 木 | 淋（水木）→ 24伤；洪（水水木）→ AOE 36伤 |

### 双重加成（相生×相克）

相生×3 与相克×1.5 共同作用，例：
- 焚（木生火×3）vs 金系（火克金×1.5）：floor(18×3×1.5)=81 AOE伤

### 五行相生的生态逻辑（世界观参考）

| 母 | 子 | 逻辑 |
|----|----|----|
| 木 | 火 | 木材是燃烧的燃料 |
| 火 | 土 | 火燃烧后灰烬化为尘土 |
| 土 | 金 | 矿石蕴藏于泥土之中 |
| 金 | 水 | 金属融化为液体 |
| 水 | 木 | 水能滋润树木生长 |
